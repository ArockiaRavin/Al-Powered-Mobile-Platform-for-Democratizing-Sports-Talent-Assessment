# Backend - Democratizing Sports Talent Assessment

## Overview

This backend powers the **Democratizing Sports Talent Assessment** platform. It provides secure REST APIs for user authentication, player management, sports performance assessment, AI-powered analysis, and administrative operations.

## Features

* User Registration & Login
* Player Profile Management
* Sports Talent Assessment APIs
* AI-Based Performance Analysis
* Video Upload Support
* Admin Dashboard APIs
* Secure Authentication
* Database Integration

## Technology Stack

* Python
* Flask
* Flask REST API
* SQLAlchemy
* MySQL / SQLite
* OpenCV
* MediaPipe
* TensorFlow
